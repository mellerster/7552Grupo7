Descripción detallada

Contiene la lista de mensajes pertenecientes a una conversación.

Se mapea a un JSON de la siguiente forma: 

 {
   Status: "",
   Token: "",
   IDConversacion: "",
   IDUsuario: "",
   Mensajes: [
               <MensajeDTO>,
               <MensajeDTO>,
               ...
             ]
 }
