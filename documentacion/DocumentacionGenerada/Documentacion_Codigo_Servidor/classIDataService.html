virtual unsigned int IDataService::StartSession ( std::string  nombreUsuario,
std::string  password 
)
pure virtual

Comienza la sesión del usuario si este ya esta registrado en el sistema.

Chequea si el nombre de usuario y el password estan en la base de datos; Si lo estan se genera una nueva sesion asociada al usuario, si ya existia una sesion activa se la reemplaza, y devuelve un token que representa esa sesión.

Si el nombre de usuario y password NO estan en la base de datos se devuelve un token nulo.

Parámetros
[in]nombreUsuarioEl nombre de usuario del cliente.
[in]passwordEl password asociado al nombre de usuario.
Devuelve
Un token que representa la nueva sesión del usuario, "0" si no.
