Descripción detallada

Contiene los datos de estado de un usuario conectado.

Se mapea a un JSON de la siguiente forma: 

 {
   Status: "",
   Nombre: "xxx",
   Estado: "yyy",
   Foto: "<Base64>",
   UltimaActividadHora: "23:59",
   UltimaActividadFecha: "2015/05/08"
 }
