Descripción detallada

Contiene los ultimos datos de una conversación.

Se mapea a un JSON de la siguiente forma: 

 {
   Status: "",
   Token: "",
   IDConversacion: "",
   UltimoMensaje: "",
   UltimoMensajeLeido: "",
   Participantes: [
       "<userID>",
       "<userID>"
       ]
 }
