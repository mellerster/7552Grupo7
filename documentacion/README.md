# Documentación Técnica del Mensajero

## Descripción

Esta carpeta contiene la documentación tecnica de todo el sistema *MensajerO*

## Dependencias

Para poder generar la documentación es necesario

1. [sphinx] (http://sphinx-doc.org "Sitio oficial de sphinx")

## Compilación

Solo se proveen los fuentes de la documentación, por lo que es necesario compilarla, para ello se debe ejecutar en la linea de comando:

* `make html`

Para generar la dcumentación en formato HTML.
