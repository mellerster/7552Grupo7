Descripción detallada

Contiene todos los datos relacionados con los pedidos de login de los clientes.

Se mapea a un JSON de la siguiente forma: 

 {
   Token: 0,
   Status: "",
   NombreUsuario: "pepe",
   Password: "123"
 }
